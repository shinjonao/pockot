## 1. 日本語 (ja)

# プライバシーポリシー

pockot（以下「本アプリ」）は、ユーザーのプライバシー保護を最優先事項として設計されています。

## 1. データのローカル処理
本アプリで行われる音声認識、テキスト翻訳、および画像解析は、すべてユーザーの端末内（オンデバイス）で処理されます。音声データ、入力テキスト、画像データが外部のサーバーに送信されたり、収集されたりすることはありません。

## 2. 外部通信
本アプリは、以下の目的においてのみ外部通信を行います：
- 翻訳モデルおよび音声認識モデルのダウンロード
- アプリのアップデート情報の確認
- 1日の翻訳利用制限数の正確な判定（端末の時刻改ざんによる不正利用を防ぐ目的で、信頼できる外部サーバーから標準時刻情報のみを取得します。個人を特定するデータや固有の端末識別子は一切送信・収集されません。なお、プレミアムプラン購入者に対しては、この時刻情報取得のための通信は一切行われません）

これらの通信に、ユーザーの個人情報や翻訳内容が含まれることはありません。

## 3. データの保存
ユーザーの翻訳履歴は端末内にのみ保存されます。履歴の削除は履歴画面からいつでも行うことが可能です。

## 4. 統計情報の収集
本アプリの品質向上のため、個人を特定しない形でのクラッシュレポートや利用統計（例：どの機能が使われたか）を収集する場合があります。

## 5. 免責事項
本アプリが提供する翻訳結果はAIによって生成されたものであり、その正確性や妥当性を保証するものではありません。

pockot (hereinafter "the App") prioritizes the protection of user privacy.

The App performs external communication only for the following purposes:
